什么是动词

简单来说，描述动作的词就叫作“动词”。

动词比较复杂，所以这里单独用一个章节来简单描述动词的各种特性。先有个了解，在后继的学习中才会觉得轻松。

注意：以下内容中，点词查词，点图听例句。

1. 及物与不及物

及物表示动作有承受者，称为及物动词（ transitive verb ，简称 vt）。

例： I eat an apple .

其中动词 eat 为及物动词，因为 apple 是动词的承受者。

不及物表示动作没有承受者，称为不及物动词（ intransitive verb ，简称 vi）。

例： I run .

其中动词 run 为不及物动词，因为动作没有承受者。

有很多动词既是及物动词同是也是不及物动词，但同一动词做及物或不及物时，单词含义会有一些差异，可根据要表达的内容需要来使用。

2. 动词的时态

时态就是动作发生的时间与当前的状态。

发生的时间可分为：过去、现在、将来、过去将来（我本来打算做什么）；

当前的状态可分为：已完成、进行中、未发生、习惯性动作。

时间有4种，状态有4种，它们两两之间可以进行组合，所以英语中共有16种可能的时态组合。如：过去完成时等。

要描述英语句子中的不同时间与状态都是通过对动词不同时态的变形来完成的，由此可见动词的重要性与复杂性。

3. 动词不定式

动词不定式就是由动词不定式符号to加动词原形构成了一种非谓语动词结构（也有动词可以不带to）。

例： I want to go .

例句中有两个动词 want 与 go 。其中 want 是主语的直接动作，可以通过对它的变形来展现句子的时态； 而 go 是主语想做的动作，但不确定什么时候做，也就无法确认时态，所以要用到动词不定式。

不清楚什么是谓语？将在下一节中进行说明，这里只须要有个印象即可。

4. 动词变形简单举例

一个动词可以进行变形，在不同的句子和时态环境中需要使用到不同的动词形态来表达到相应的含义。

本节先粗讲一下动词有哪些形态，在什么情况下会用到哪种动词变形，后面会通过实例详细说明。

以动词 go 为例，各变形如下：

1. 动词原型（go）；

2. 动词过去式（went）；

3. 动词现在分词（going）；

4. 动词过去分词（gone）；

5. 动词单数三人称（goes）。

所以记忆一个动词，你须要对动词的5种形态都进行了解，才算是掌握了。不过也不要担心，大部分动词都是有规律可循的。

5. 后继学习计划

本节只是对动词的相关特性做了一个大致的介绍，还有很有细节的地方，会在后面结合实际句子进行详细说明。

为了锻炼英语思维，在当前的初学阶段，尽量多锻炼英语思维。即大家尽量在看单词、句子的时候直接通过图片来联想含义，尽量不要通过中文去转换！开始可能比较困难，但如果养成习惯后，对英语的学习帮助是非常大的。
