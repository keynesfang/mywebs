句子的基本成分
在纲要第三点提到了“要从最简单的句子入手，然后不断扩充句子的成分”来学习语法，本文就来介绍一下什么是最简单的句子。
1. 主语 + 谓语（主谓结构）
I主语 escape谓语. 我逃脱了。
这应该是最简单的基本结构了，就两个单词，一个主语，一个谓语。
2. 主语 + 谓语 + 宾语（主谓宾结构）
I主语 eat谓语 an apple宾语. 我吃一个苹果。
3. 主语 + 谓语 + 间接宾语 + 直接宾语（主谓双宾语结构）
I主语 buy谓语 you间宾 a book直宾. 我给你买了本书。
一般来说指人为间接宾语，物品名词为直接宾语。以此例来说，“买”的是“书”，是动词的直接作用对象，所以书是“直接宾语”。而“你（you）”是“书”的接收者，与动词“买”没有直接关系，所以为间接宾语。
3. 主语 + 谓语 + 宾语 + 宾语补足语（主谓宾宾补结构）
I主语 named谓语 the dog宾语 Candy宾补. 我给条狗取名叫坎迪。
在谓语动词之后接了宾语，但句义依旧不完整，需要再加另外的成分补充说明宾语的意义，这个成分就是宾语补足语。
2. 主语 + 系动词 + 表语（主系表结构）
He主语 looks谓语 handsome表语. 他看起来真帅。
把主语和表语联系起来的动词，叫系动词（Linking Verb）。它虽然本身有词义，但不能单独用作谓语，其后必须跟表语，构成系表结构说明主语的状况、性质、特征等情况。 系动词最常见的是be，还有“五感”feel、look、sound、smell、taste和其它动词如become、grow...等。
