句子的基本结构
上一节讲了句子的基本结构。其中涉及到了主语、谓语、宾语、动词、系动词等的组合情况。

主语 + 谓语
I主语 escape谓语. 我逃脱了。
【主语 + 谓语（动词）】表示主语做了某个动作，构成了最简单英语句子。注意，这里的动词必须是不及物动词。
主语 + 谓语 + 宾语
I主语 eat谓语 an apple宾语. 我吃一个苹果。
这个句型比上面多了一个宾语，表示主语对某个事物做了某个动作。这个宾语实际上就是谓语（动词）的接受者。所以，这里的动词必须是及物动词。
主语 + 谓语 + 间接宾语 + 直接宾语
I主语 buy谓语 you间宾 a book直宾. 我给你买了本书。
这个句型相比上面，把宾语分成了直接宾语与间接宾语，表示谓语（动词）影响了两个事物，一个是直接影响的，一个是间接影响的。 以此例来说，“买”的是“书”，是动词的直接作用对象，所以书是“直接宾语”。而“你（you）”是“书”的接收者，与动词“买”没有直接关系，所以为间接宾语。
主语 + 谓语 + 宾语 + 宾语补足语
I主语 named谓语 the dog宾语 Candy宾补. 我给条狗取名叫坎迪。
这个句型相比主谓宾句型，多了一个宾语补足语，说明在主谓宾的情况下，句义还不完整，需要再加另外的成分补充说明宾语的意义，这个成分就是宾语补足语。
主语 + 系动词 + 表语
He主语 looks谓语 handsome表语. 他看起来真帅。
前面【主语 + 谓语（动词）】的句型表达了主语做了什么动作及对其他事物的影响，而【主语 + 系动词】构成系表结构则用来表示主语的身份、类别、特征、状态等情况。把主语和表语联系起来的动词，叫系动词。
总结
以上介绍了英语中最基本的5种句型结构。在后面的内容中，我们将基于这些结构对英语句子的构成进一步进行剖析。俗话说“读书百遍，其意自现。”如果你是初学者，或记忆力不是很好，可以没事时来多看几遍，也许每次都会有新的收获。
